# Applicant Analysis Report Template

Use this template when the user asks for a complete school-positioning or application-strategy report.

If the user asks for an HTML, printable, visual, or designed report, also read `html-report.md`.

## Required Order

1. 分析结构
2. 背景定位与核心判断
3. 地区与专业策略
4. 项目分层与申请组合
5. 提升方案与文书主线
6. 相似背景申请参考与资料来源

## Section Guidance

### 1. 分析结构

Briefly list what the report will evaluate. Mention:
- academic background
- language score
- internships and competitions
- target regions
- target disciplines
- budget/ranking constraints
- official programme requirements
- multi-agency public case references

### 2. 背景定位与核心判断

Convert the profile into structured signals:
- institution tier
- GPA band
- IELTS band
- major fit
- internship fit
- competition/project evidence
- target-region fit
- budget pressure

Give a concise assessment. Use labels such as:
- stretch
- moderate stretch
- target
- safer

Explain the logic. Do not stop at a label. Each major claim must be grounded in one of:
- official programme requirements
- public agency cases
- public agency commentary
- the user's stated profile
- clearly labelled inference from those sources

If public cases are sparse, say so and lower confidence.

### 3. 地区与专业策略

Separate Hong Kong and Singapore, then compare target disciplines. Discuss:
- competitiveness
- tuition pressure
- IELTS threshold
- programme fit
- employment fit
- application timing

Ground region and discipline strategy in public agency cases, public agency commentary, official programme pages, and the user's preferences. Avoid generic statements about a country or discipline unless supported.

Compare the user's requested tracks. For language/communication/marketing applicants, distinguish:
- applied linguistics
- English studies for professional communication
- translation
- communication / IMC
- marketing
- marketing analytics
- social science pivot

### 4. 项目分层与申请组合

Use reach / target / safer. For each project or project type, explain:
- why it fits
- the main risk
- what would improve the odds
- cost sensitivity if known
- the evidence basis: official requirement, public case signal, agency commentary, or inference

Include application portfolio advice:
- how many reach, target, and safer programmes
- which region should be primary
- which expensive programmes should be limited

### 5. 提升方案与文书主线

Make the advice operational. Include:
- language score target
- portfolio or project suggestions
- internship packaging
- course or skill gaps
- recommendation-letter positioning
- transcript/GPA risk handling

Build coherent story-line frameworks, not full essay drafts unless asked. For language/marketing profiles, a strong umbrella line is often:
"cross-cultural communication + media writing + agency/client management + consumer/brand insight."

Avoid generic claims such as "I am passionate about marketing." Tie each claim to evidence.

For essay and improvement advice, explicitly connect each suggestion to the applicant's real profile or to public agency commentary/cases. Do not invent a theme that the profile cannot support.

For multi-track applicants, provide several direction-specific story frameworks by reading `essay-feedback-rubric.md`. Cover the applicant's actual domains, not only humanities. Possible tracks include:
- applied linguistics / professional English
- communication / IMC / media
- marketing / brand management
- marketing analytics / consumer insight
- language education / TESL
- finance / accounting / investment
- management / strategy / entrepreneurship
- business analytics / data science / information systems
- computer science / software engineering / AI
- engineering / built environment / industrial technology
- science / research / lab-based programmes
- social science / public policy / international affairs
- law / compliance / IP / governance

Include an execution timeline inside this section when the application season is in the future.

### 6. 相似背景申请参考与资料来源

Keep this exact table format:

| 来源 | 相似案例 | 背景信号 | 录取结果 | 对你的参考 |
|---|---|---|---|---|

Rules:
- Prefer multiple agency sources.
- Include exact school matches when found.
- Add adjacent cases from other agencies when exact matches are concentrated in one source.
- Keep facts concise.
- Make the "对你的参考" column analytical and specific.
- Distinguish exact matches, adjacent matches, and weak references.
- Do not use a case as proof. Use it as directional evidence only.

After the table, put references at the end. Include:
- official programme pages
- public case pages
- source caveats

State when 2028-intake information is not yet available and current pages are being used as a proxy.
